## 시간복잡도
- 알고리즘을 위해 필요한 연산의 횟수
- 일반적으로 알고리즘 문제에서의 복잡도란 이 시간복잡도를 의미한다
- 1초의 시간제한을 넘지않으려면 연산횟수를 10억 이하로 잡아야한다

<br>

## 공간복잡도
- 필요한 메모리의양
- 125MB ~ 512MB, 즉 데이터의 갯수가 천만단위가 넘어가지 않도록 해야한다
```python
# 시간측정에 쓰이는 코드
import time
st = time.time()
...
et = time.time()
print(et-st)
```

# 그리디
- 당장 좋은것을 고르는 방법
- 그리디는 이해는 기본, 암기하면 더 완벽한 알고리즘이다
- 주로 "가장 큰 순서대로~, 가장 작은 순서대로~" 로 정렬문제와 같이 등장한다
