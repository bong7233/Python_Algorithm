# 알고리즘 복잡도
- 일반적인 코딩테스트 문제에서 말하는 복잡도란 시간복잡도를 의미한다
- bigO  
![timecomplexity](../img/bigO.jpeg)  
  

- 선형 시간 알고리즘 O(n)
  - n개의 수를 하나씩 모두 살펴보는경우  


- 로그 시간 알고리즘 O(log n)
  - 정렬된 n개의 수를 이진탐색하는경우  


- 이차 시간 알고리즘 O(n^2)
  - n번의 작업을 두번하는 경우
  - 삽입정렬, 반복문 내부의 반복문  


- 낮은 복잡도를 가지는 정렬 알고리즘 O(n log n)
  - 병합정렬
